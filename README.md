# WeedClient

A visual client mod for Minecraft 1.21.4 using Fabric.

## Features

- 🟢 Green gradient visual aesthetic
- ⌨️ Left Shift to open visual menu
- 🎨 Customizable visuals

## Installation

1. Install [Fabric Loader](https://fabricmc.net/)
2. Place the mod jar in your mods folder
3. Launch Minecraft 1.21.4

## Controls

- **Left Shift** - Open WeedClient menu

## Building

```bash
./gradlew build
```

The compiled jar will be in `build/libs/`

## Development

1. Clone this repository
2. Run `./gradlew build` to set up the workspace
3. Run `./gradlew runClient` to test the mod
